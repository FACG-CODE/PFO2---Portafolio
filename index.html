Proyectos

Sistema de Gestion de Consultorios

Se trata de un sistema informático para escritorio, con base de datos local, el mismo cuenta con distintas funcionalidades como el registro de empleados administrativos, personal médico y pacientes, la gestión de turnos, el registro del proceso de atención médica, y la liquidación de honorarios para el personal médico contratado.

Sistema de Gestion de Consorcios

Sistema informático destinado inicialmente para escritorio, se trata de un software con funcionalidades que automatizan las tareas de un Administrador de Consorcios. El mismo cumple con la actual legislación que regula la actividad en CABA. Sus funciones principales son, registro de personas con diferenciación de roles, registro de gastos del consorcio, la generación de balances de expensas y recibos de pago, el registro de pagos de propietarios y la visualización del estado contable para su analisis.
